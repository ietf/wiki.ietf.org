The following diagrams (in ascii-art, and aasvg generated SVG) are to be used to explain various kinds of composite attesters.

# Visual Atom

* [A Conveyer](conveyer.md) is a system component that produces or relays Conceptual Messages.

* [An Attester](attester.md) is a special kind of Conveyer which produces Evidence.

* [An interface](interface.md) is connected to a Node (such as an Attester) and outputs a RATS Conceptual Messages.

* [A Dependant](dependson.md) connector describes a chain of trust between two adjacent Attesters within a layered attester arrangement.

* [A Collection](collection.md) connector describes the collection of Conceptual Messages.
