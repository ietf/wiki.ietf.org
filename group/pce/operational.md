# Operational Considerations for PCE WG Drafts

All *applicable* PCE WG drafts need to include an **"Operational Considerations"** section, following the guidance in [draft-ietf-opsawg-rfc5706bis](https://datatracker.ietf.org/doc/draft-ietf-opsawg-rfc5706bis/). 

In the past, the PCE WG has used the "Manageability Considerations" section from [RFC 6123](https://datatracker.ietf.org/doc/html/rfc6123#section-2.4) as a template. Going forward, authors should instead follow the updated guidance in 5706bis. Note that RFC 6123 is already marked Historic.
